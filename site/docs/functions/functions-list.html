Listing existing functions

You can list all existing functions by using the kn faas list command to list the names of functions. However, because functions on OpenShift Serverless are deployed as Knative services, you can also list them using kn service list.

Procedure

  • List function names:

    $ kn faas list
    Example output
    fn.example.io

  • List functions as Knative services:

    $ kn service list
    Example output
    NAME            URL                                                                                       LATEST                AGE   CONDITIONS   READY   REASON
fn-example-io   http://fn-example-io-default.apps.ci-ln-g9f36hb-d5d6b.origin-ci-int-aws.dev.rhcloud.com   fn-example-io-gzl4c   16m   3 OK / 3     True
